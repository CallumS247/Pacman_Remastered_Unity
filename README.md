# Pacman_Remastered_Unity
Pacman with some new features and upgrades. Assets made by myself. Unity Ver. 2022.3.54

New Features include:
  - Save files and profile select
  - Profile screen with stats and achievements
  - New colour schemes for customization
  - 2nd Map with alternative layout
  - New hazards that will rotate each wave
  - Abilities + Loadout system
  - Classic/Remastered game modes
